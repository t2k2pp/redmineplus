🎯 RedminePlus

Redmineチケット可視化ダッシュボード

RedmineのAPIを使用してチケットデータを取得し、
インタラクティブなグラフとレポートで可視化します

🚀 デモ・実行方法

以下のオンラインサービスでアプリを実行できます：

📊 Streamlit Community Cloudで実行

または、ローカル環境でセットアップして実行してください

📊 豊富な可視化機能

  • ステータス・優先度・担当者別の分析グラフ
  • スケジュール・期限分析とガントチャート
  • 工数分析と進捗管理
  • 時系列分析

⚠️ アラート機能

  • 期限超過チケットの自動検出
  • 期限間近チケットの事前通知
  • リアルタイムでのステータス監視

📋 チケット詳細管理

  • インタラクティブなチケット一覧
  • 詳細情報とコメント履歴表示
  • ページネーション対応

📄 エクスポート機能

  • チケット一覧のCSVダウンロード
  • 個別チケットのPowerPoint帳票出力
  • 問題管理票形式の自動生成

💻 ローカル実行手順

1. リポジトリをクローン

git clone https://github.com/t2k2pp/redmineplus.git
cd redmineplus

2. 依存関係をインストール

pip install -r requirements.txt

3. アプリケーション起動

streamlit run streamlit_app.py

4. ブラウザでアクセス

http://localhost:8501
